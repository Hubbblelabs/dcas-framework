# DCAS Assessment Web App

This is a Next.js application implementing the Career Behaviour Mapping Framework (CBMFâ„¢) / DCAS Assessment.

## Features
- **Assessment**: 30-question behavioral assessment.
- **Results**: Real-time scoring and visualization of Driver, Connector, Anchor, and Strategist traits.
- **Design**: Modern, responsive UI using Tailwind CSS and Shadcn/UI.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open the app**:
    Navigate to [http://localhost:3000](http://localhost:3000).

## Project Structure
- `src/app`: App Router pages.
- `src/components`: Reusable UI components.
- `src/data`: Extracted assessment questions (`questions.json`).
- `src/types`: TypeScript definitions.

## Tech Stack
- Next.js 16
- Tailwind CSS
- Shadcn/UI
- Recharts
- Lucide React
